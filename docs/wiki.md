# DataNavigator Wiki

Deze wiki geeft een overzicht van belangrijke onderdelen van de applicatie.

## Connection Manager
Hiermee beheer je eenvoudig verbindingen naar externe databases of Power BI modellen. De manager ondersteunt het testen, opslaan en bewerken van verbindingen.

## Catalog Execution
Start en monitor het catalogeren van databronnen. Elke run wordt gelogd zodat je later historische runs kunt terugkijken inclusief details uit de logbestanden.
